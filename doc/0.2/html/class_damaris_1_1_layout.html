Detailed Description

Layouts are objects that describe the representation of data in memory. For example "an 3D array of integers which extents are 4,2 and 16". Actually we choosed to represent layouts as in Fortran, using a starting index and an ending index rather than a size. This way allows plugins to potentially know that the variable is a chunk of a bigger global array.
