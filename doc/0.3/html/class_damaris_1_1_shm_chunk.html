void * Damaris::ShmChunk::data (  )  [virtual]

Retrieves a pointer to the data.

Returns:
A pointer to where the data is stored, or NULL if the data has been removed by this object using remove().
Warning:
If several instances of ShmChunk exist (possibly in different processes) and the program calls "remove" on one of them, the other will be inconsistant. If an instance of ShmChunk is deleted without removing the shared memory region before, the region will be lost. Thus make sure there is always eventually exactly one ShmChunk instance attached to a region.
