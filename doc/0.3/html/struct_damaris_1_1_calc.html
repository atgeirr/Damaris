Detailed Description

template<typename Iterator, typename SymTable>
struct Damaris::Calc< Iterator, SymTable >

Calc is a structure containing Boost.Spirit parser elements required to compute a list of values from a string-based representation of a Layout. The input is a comma-separated list of arythmetic expressions using +, -, *, /, % and identifier whose value is defined in a SymTable mapping.
